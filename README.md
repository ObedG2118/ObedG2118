## 🏆 Proyectos Destacados

### 🐑 VITRA - Sistema de Diagnóstico Veterinario Asistido por IA
Aplicación móvil multiplataforma diseñada para el diagnóstico temprano de anemia en ovinos, automatizando la evaluación clínica mediante la técnica FAMACHA.

* **Tecnologías:** Flutter, Dart, Python, TensorFlow Lite, SQLite.
* **Aportes Técnicos Clave:**
    * Implementación de arquitectura *Offline-First*, garantizando la operatividad del sistema en zonas rurales sin acceso a internet.
    * Integración de modelos de *Visión por Computadora* optimizados para la ejecución local en dispositivos móviles, eliminando la latencia de la nube.
    * Gestión eficiente de bases de datos locales con sincronización asíncrona.
